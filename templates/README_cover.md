# 封面图说明

本目录用于存放微信公众号草稿的封面图。

## 使用方法

1. 准备一张封面图片（JPG 或 PNG 格式）
2. 推荐尺寸：900 x 383 像素（微信公众号图文消息封面图标准尺寸）
3. 将图片重命名为 `default_cover.jpg` 或 `default_cover.png`
4. 保存到 `templates/` 目录

## 封面图要求

- 格式：JPG、PNG
- 大小：不超过 2MB
- 推荐尺寸：900 x 383（宽高比约 2.35:1）
- 图片内容：建议使用与 AI、科技相关的配图

## 注意事项

- 如果不提供封面图，测试代码会使用占位 media_id，实际创建草稿时会失败
- 封面图会被上传到微信公众号素材库，占用素材库空间
- 上传后的封面图可以重复使用（通过 media_id）
